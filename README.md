# 🌐 Portfolio Website (Full Stack)


This is a full-stack portfolio website built using **Node.js, Express, and PostgreSQL** with a simple frontend using HTML, CSS, and JavaScript.

---

## 🚀 Features

- 📩 Contact form to collect user messages
- 🗄️ Messages stored in PostgreSQL database
- 🌐 Backend deployed on Render
- ⚡ REST API for handling requests
- 🔒 Secure database connection using environment variables
- 🔄 Continuous Integration using GitHub Actions

---

## 🛠️ Tech Stack

**Frontend:**
- HTML
- CSS
- JavaScript (Fetch API)

**Backend:**
- Node.js
- Express.js

**Database:**
- PostgreSQL (Render)

**DevOps:**
- GitHub Actions (CI)
- Render (Deployment)
